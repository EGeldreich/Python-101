# 🐍 Python → Data Analyst — Parcours d'apprentissage complet

> Mon programme structuré pour apprendre Python from scratch et devenir opérationnel en data analyse : manipulation de données, visualisation, automatisation de rapports.

---

## 🎯 Objectif

Ce dépôt contient l'intégralité de mon parcours d'apprentissage Python orienté **Data Analyst**, construit autour d'un principe simple : chaque notion est expliquée, illustrée par un exemple commenté ligne par ligne, puis mise en pratique sur des cas concrets (ventes, clients, chiffres d'affaires, dates...).

**À l'issue de ce parcours, je serai capable de :**

- Écrire des scripts Python propres et organisés
- Charger, nettoyer et transformer des données avec **Pandas**
- Produire des visualisations claires avec **Matplotlib** et **Seaborn**
- Automatiser des rapports et exports Excel/CSV

## 📋 Suivi de progression

Chaque case cochée représente mon avancée actuelle.

---

### Module 1 — Les fondations de Python

> _Durée estimée : 3 à 5 heures_

- [x] Installer Python, VS Code et Jupyter Notebook
- [x] Comprendre les variables et les types de données (`int`, `float`, `str`, `bool`)
- [x] Effectuer des opérations de base et comprendre les priorités
- [x] Utiliser l'affichage formaté (`print`, f-strings)
- [x] ✅ **Exercices module 1 complétés** — Premières données de ventes

---

### Module 2 — Structures de données et logique

> _Durée estimée : 4 à 6 heures_

- [ ] Créer et manipuler des **listes** (séries de données)
- [ ] Utiliser les **dictionnaires** (données structurées type fiche client)
- [ ] Connaître les tuples et les ensembles
- [ ] Écrire des conditions `if / elif / else`
- [ ] Maîtriser les boucles `for` et `while`
- [ ] ✅ **Exercices module 2 complétés** — Analyse d'un portefeuille clients

---

### Module 3 — Fonctions et organisation du code

> _Durée estimée : 3 à 4 heures_

- [ ] Créer et appeler des **fonctions**
- [ ] Utiliser les paramètres, valeurs par défaut et valeurs de retour
- [ ] Comprendre la portée des variables (scope)
- [ ] Importer et utiliser des **modules** (`math`, `random`, `datetime`...)
- [ ] Gérer les erreurs avec `try / except`
- [ ] ✅ **Exercices module 3 complétés** — Calculs automatisés de CA et marges

---

### Module 4 — Pandas, le cœur du data analyst

> _Durée estimée : 6 à 8 heures_

- [ ] Comprendre les objets `Series` et `DataFrame`
- [ ] Charger des fichiers **CSV et Excel** avec Pandas
- [ ] Explorer un dataset (`head`, `info`, `describe`, `shape`)
- [ ] Sélectionner, filtrer et trier des données (`loc`, `iloc`, conditions)
- [ ] Nettoyer les données (valeurs manquantes, doublons, types)
- [ ] Grouper et agréger (`groupby`, `pivot_table`, `agg`)
- [ ] Fusionner des tableaux (`merge`, `concat`)
- [ ] ✅ **Exercices module 4 complétés** — Analyse complète d'un fichier de ventes

---

### Module 5 — Visualisation des données

> _Durée estimée : 4 à 5 heures_

- [ ] Créer des graphiques de base avec **Matplotlib** (courbes, barres, camemberts)
- [ ] Utiliser **Seaborn** pour des graphiques statistiques (boxplot, heatmap, pairplot)
- [ ] Savoir choisir le bon type de graphique selon les données
- [ ] Personnaliser titres, axes, couleurs et légendes
- [ ] ✅ **Exercices module 5 complétés** — Dashboard visuel d'un rapport mensuel

---

### Module 6 — Automatisation et projet final

> _Durée estimée : 5 à 6 heures_

- [ ] Lire et écrire des fichiers (CSV, Excel, JSON)
- [ ] Automatiser la génération de rapports avec Pandas + `openpyxl`
- [ ] Manipuler les dates et séries temporelles (`datetime`, `resample`)
- [ ] ✅ **Mini-projet final complété** — Pipeline complet d'analyse de données
